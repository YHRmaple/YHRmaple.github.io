html元素分类

作者: 杨浩然 日期: August 17, 2020  10:48:45 分类: 前端

之前对于 HTML 的元素标签分类经常搞混，所以在这里做一个总结。大致可分为三种类型：块级元素（block），内联元素（又称行内元素，inline），内联块级元素（inline-block）

块级元素

特点

  1. 每个块级元素独占一行，后面的元素会另起一行。
  2. 可以设置高度，宽度，行高，边框，内外边距。
  3. 宽度默认为父容器的100%。

举例

标签 作用
<address> 定义地址
<caption> 定义表格标题
<dd> 定义列表中定义条目
<div> 定义文档中的分区或节
<dl> 定义列表
<dt> 定义列表中的项目
<fieldset> 定义一个框架集
<form> 创建 HTML 表单
<h1>……<h6> 定义标题
<hr> 创建一条水平线
<legend> 为 fieldset 元素定义标题
<li> 定义列表项目
<noframes> 为那些不支持框架的浏览器显示文本，于 frameset 元素内部
<noscript> 定义在脚本未被执行时的替代内容
<ol> 定义有序列表
<ul> 定义无序列表
<p> 标签定义段落
<pre> 定义预格式化的文本
<table> 标签定义 HTML 表格
<tbody> 标签表格主体（正文）
<td> 表格中的标准单元格
<tfoot> 定义表格的页脚（脚注或表注）
<th> 定义表头单元格
<thead> 标签定义表格的表头
<tr> 定义表格中的行

转换

可通过设置{display:block}将元素转换为块级元素

内联元素

特点

  1. 和其他元素在一行

  2. 元素的高度，宽度，行高，顶和底的边框，顶和底的内外边距不可改变（本人在chrome测试，顶和底的边框以及内边距如果设置了颜色，会显示出来，但不占用实际位置，而且会挡住上方的元素，但不会挡住下方的元素）

  3. 元素的宽度就是它所包含的文字或图片的宽度，不可改变。

举例

标签 作用
<a> 锚
<br> 换行
<i> 斜体文本效果
<span> 组合文档中的行内元素
<strong> 语气更强的强调的内容
<cite> 引用进行定义
<code> 定义计算机代码文本
<em> 定义为强调的内容
<label> 为 input 元素定义标注
<q> 定义短的引用
<small> 呈现小号字体效果
<var> 定义变量

转换

可通过设置{display:inline}将元素转换为内联元素

内联块级元素

特点

  1. 和其他元素在一行上
  2. 可以设置高度，宽度，行高，边框，内外边距。

举例

标签 作用
<img> 定义图像
<input> 定义输入域
<select> 定义下拉选项列表

转换

可通过设置{display:inline-block}将元素转换为内联块级元素

在实际开发过程中，要根据需求灵活转换
